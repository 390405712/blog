# Git
## 常用命令
- 下载项目：git clone url
- 新建git配置：git init
- 查看状态：git status
- 删除某个文件：git checkout xxx
- 切换分支：git branch
- 新建分支/切换分支：git checkout -b xxx / git checkout xxx
- 对比文件：git diff
- 合并分支：git merge xxx
- 加入所有文件 ：git add .
- commit：git -commit -m 'xxx'
- push：git push origin master
- pull：git pull origin master
