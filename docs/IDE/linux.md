# Linux命令
1. rm xxx - 表示删除某个文件
2. rm -rf!!! xxx - 表示循环删除某个文件夹（禁用）
3. pwd - 输出当前文件夹
4. mkdir - 创建某个目录
5. cd - 进入某个目录
6. ls -al - 能把所有文件和权限打出来
7. touch - 创建文件
8. cat a.txt 查看某个文件
9. vi a.txt - 编辑文件
 1. vi之后按下i键，进入insert模式
 2. 写完之后 ，先按下：，按q表示推出（一般不允许你退出）
 3. 按下q!强制退出
 4. 按下wq，保存退出
10. echo xxx > a.txt - >输入  >> 表示叠加
